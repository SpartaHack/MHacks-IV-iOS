MHacks-iOS
===========

Official iOS App for MHacks IV, forked from HSHacks

To build this, you'll need to make a file in .../MHacks-iOS/MHacks called appKeys.plist with the following fields:

* parseAppId
* parseClientKey
* twitterConsumerKey
* twitterConsumerSecret
